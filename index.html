数据类型

要分析==，首先从javascript的变量类型说起。

javascript有9种数据类型：

  • Undefined
  • Null
  • Boolean
  • String
  • Number
  • Object
  • Reference
  • List
  • Completion

其中 Reference，List，Completion只是作为执行的中间结果，因此我们关注的是前6种数据类型，分为三类：

  • Undefined，Null
  • Boolean，String，Number
  • Object
